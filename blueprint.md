# TM(견인전동기) 현황 관리 시스템 - Excel Edition

## 프로젝트 개요
보내주신 '1번 TM 현황 DB' 엑셀 파일을 기반으로, 사용자가 익숙한 **엑셀 스타일의 UI**를 가진 웹 기반 데이터 관리 프로그램입니다. 단순히 데이터를 보여주는 것을 넘어 실시간 위험도 계산과 데이터 입력 기능을 제공합니다.

## 디자인 컨셉 (Excel Style)
*   **Formula Bar:** 선택된 셀의 내용을 보여주는 영역 (시각적 효과).
*   **Central Grid:** A, B, C... 열 헤더와 1, 2, 3... 행 번호가 있는 데이터 그리드.
*   **Bottom Sheet Tabs:** 엑셀의 워크시트와 동일하게 하단 탭을 통해 10개의 시트 전환 가능.
*   **Cell Selection:** 클릭 시 셀 테두리 강조 효과.

## 주요 워크시트 구현 (10개)
1.  **사용방법:** 시스템 가이드 및 원칙 설명.
2.  **대시보드:** 요약 통계 및 차트.
3.  **TM_MASTER:** 현재 장착/보유 현황 및 위험 점수 관리.
4.  **TM_REPLACEMENT:** 교체 이력 로그.
5.  **TM_FAILURE:** 고장 이력 상세 분석.
6.  **FAILURE_CODE:** 표준 고장 코드 정의.
7.  **DETAIL_MAP:** 원본 고장명 매핑 가이드.
8.  **RISK_RULE:** 위험도 계산 공식 참조.
9.  **INPUT_TEMPLATE:** 데이터 입력을 위한 양식.
10. **LOOKUP:** 표준 관리 목록.

## 기술적 특징
*   **Vanilla JS Grid Engine:** 순수 자바스크립트로 엑셀 그리드 감성 구현.
*   **Dynamic Rendering:** 선택된 시트에 따라 테이블 구조와 데이터를 실시간으로 변경.
*   **Excel Aesthetics:** 연한 회색 배경, 흰색 그리드 셀, 초록색 탭 강조 포인트.
